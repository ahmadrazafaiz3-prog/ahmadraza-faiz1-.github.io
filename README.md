# My Jekyll Blog Website

Welcome to my personal blog website built with **Jekyll** and hosted on **GitHub Pages**.  
This website uses the **Contrast theme** and showcases my weekly posts on web development, databases, GitHub, AI, and CSS.

- Static homepage with personal information
- Blog section with weekly posts
- Each post contains:
  - Title
  - Content
  - Relevant image
- Simple and clean design with Contrast theme
